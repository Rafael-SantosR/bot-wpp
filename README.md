# 🤖 WhatsApp Chatbot (Node.js)

Chatbot para WhatsApp desenvolvido com Node.js utilizando a biblioteca whatsapp-web.js.

## 🚀 Funcionalidades
- Envio e recebimento automático de mensagens
- Respostas baseadas em comandos
- Leitura de mensagens em tempo real
- Conexão via QR Code

## 🛠️ Tecnologias
- Node.js
- whatsapp-web.js
- qrcode-terminal
